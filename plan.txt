Problem #1: Render Free Tier Cold Start (Most Likely Cause)
Tumhara backend Render free tier par hai. Free tier me server 15 min inactivity ke baad so jata hai (sleep/spin down). Jab pehla customer QR scan karta hai:

Frontend → Backend ko request bhejta hai
Backend server soya hua hai → wake up hone me 30-50 seconds lagta hai
But frontend ka fetch call ka koi explicit timeout nahi hai — browser ka default timeout ~60-120s hai
OpenRouter ka timeout sirf 15 seconds hai (->timeout(15))
Server wake up + prompt build + OpenRouter call = easily 20-30 seconds
Agar cold start + AI model slow response mila → total time 15s se zyada → OpenRouter timeout → sabhi 5 models timeout → null return → ❌ "Review generate nahi ho paya"
Ye "kabhi kabhi" isliye hota hai kyunki: agar server already awake hai (recent traffic aaya tha) toh kaam karta hai. Agar lamba gap tha → cold start → fail.

Solution #1:
cron-job.org kya karega?
Socho tumhara Render backend ek dukaan hai 🏪

Agar 15 min tak koi customer nahi aaya → dukaan ka shutter band ho jata hai (Render server sleep)
Jab naya customer aata hai → shutter kholne me 30-50 sec lagta hai (cold start)
Customer itna wait nahi karta → chala jata hai (review fail)
cron-job.org = Ek watchman 🧑‍✈️

Ye har 14 min me tumhare server ko "hello" bolta hai (ek chhoti si ping request)
Server sochta hai "koi aaya hai" → jagta rehta hai, soya nahi kabhi
Ab jab real customer aaye → server already jaaga hua → review 1-2 sec me generate ✅
